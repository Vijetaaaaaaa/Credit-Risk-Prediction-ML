# Credit-Risk-Prediction-Ml
Machine learning project for predicting credit default risk using Logistic Regression, Random Forest, and Streamlit

## Overview
This project predicts the likelihood of credit card default using machine learning models.

## Models Used
- Logistic Regression
- Random Forest Classifier

## Features
- Data preprocessing
- Feature engineering
- Model evaluation (ROC-AUC, Precision, Recall)
- Streamlit web app UI

## Results
- Logistic Regression ROC-AUC: ~0.70
- Random Forest ROC-AUC: ~0.77

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

## Purpose
Designed as a fintech/risk analytics project to demonstrate machine learning applications in financial risk prediction.
